Welcome to the Stock Market Prediction GUI

This has been coded on Angular 5.0. As I plan to show how the visualization looks on a GUI I am writing this on Angular. Once I am done with the predictive analysis and the predictive model, I will implement that in a web framework such as Django on the back end and all the computation will happen on the backend. Right now we are using Django to compute and get the stock market's data. All the Data has then been converted to a JSON file to be retrieved by angular. All of the work has then been used as an API for Angular to display the content.
